# Blog Application 📝🚀  

A full-stack **MERN (MongoDB, Express.js, React, Node.js)** application that allows users to create, manage, and interact with blogs.  

## 🌟 Features  
### 📝 For Authors (Users):  
- ✅ Create, edit, and delete blogs.  
- ✅ Categorize blogs for better organization.  
- ✅ Secure authentication for managing blogs.  

### 💬 For Readers (Viewers):  
-  Browse blogs based on categories.  
-  Engage with blogs through comments.  
-  View blog details with a clean UI.  

### 🔍 Additional Features:  
-  Responsive & user-friendly interface.  
-  Optimized backend for smooth performance.  
-  Protected routes with authentication & authorization.  

---

## 🛠 Tech Stack  
- **Frontend:** React.js, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Token)  
- **Other Tools:** Mongoose, bcrypt.js, dotenv
